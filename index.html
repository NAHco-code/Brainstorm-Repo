Weather Dashboard

Intuitive user experience
Clean and polished user interface

A weather application that allows a user to search the weather outlook for multiple cities. Allows the user to better plan for events and travel.

User is presented with current and future conditions for the city searched, + each city searched is saved in search history. Current weather conditions reflect the temperature, humidity, wind speed, and the UV index. Future weather conditions present a 5-day forecast that displays the date, weather conditions, temperature, and humidity. Cities saved in search history render both current and future conditions when clicked.
